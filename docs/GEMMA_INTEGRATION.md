# Gemma Integration Guide

This file is kept only as a compatibility pointer.

Current model/runtime behavior is documented in [MODEL_INTEGRATION.md](MODEL_INTEGRATION.md).

Important changes from the old Gemma-only setup:

- the app uses runtime-downloaded LiteRT-LM `.litertlm` files
- the model is selected in Settings
- Gemma and Qwen variants are supported
- there is no supported GGUF/assets workflow anymore
